About the Project

Plugging into the Future: An Exploration of Electricity Consumption Patterns

This project presents an interactive analysis of electricity consumption patterns across India using Tableau dashboards embedded into a Flask web application.

The project focuses on

  • ⚡ State-wise Electricity Consumption
  • 📊 Quarterly Usage Analysis
  • 📈 Monthly Consumption Trends
  • 🌍 Region-wise Comparison
  • 🗺 Geographic Visualization
  • 📖 Interactive Tableau Story

Technologies Used
